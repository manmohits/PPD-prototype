For running the project, you need to have JDK, Oracle database.

Follow these following steps:-
----------------------------
1) Make a new user in oracle with user-id :mohit and password :mohit .
2) Import the backup.dmp file on your pc, where database is stored (using "imp userid=mohit/mohit file=backup.dmp full=y" at cmd prompt).
3) Now all you need is to double click the .jar , to run the project.
